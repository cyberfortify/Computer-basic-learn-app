# 💻 CompLearn – Interactive Computer Learning Platform

CompLearn is a modern, interactive web-based learning platform designed to help beginners learn computer fundamentals in an engaging way. The platform combines visual lessons, quizzes, progress tracking, notes, bookmarks, and hands-on practice tools into a single responsive application.

---

## 📖 Overview

This project is built using pure HTML, CSS, and JavaScript without any external frameworks. It provides an interactive learning experience through beautifully designed UI components, animations, quizzes, and practice exercises.

Whether you're a beginner learning computer basics or someone revising fundamental concepts, CompLearn offers an easy and enjoyable learning experience.
